User Manual

  • Target: End users
  • Applet: Unity WebGL

This section describes the controls and usage instructions for the interactive applet.

Main controls

  • Left click: add or select a point
  • Drag: move a selected point
  • Shift + Click: alternate action (e.g. second endpoint)
  • Delete / Backspace: remove the selected element
  • R: reset the scene

Modes

  • Points: creation and manipulation of points
  • Segments: construction of line segments
  • Rays / reflection: visualization of reflected rays
  • Transformations: translation, rotation, scaling

For a live demonstration, visit the Applet.
